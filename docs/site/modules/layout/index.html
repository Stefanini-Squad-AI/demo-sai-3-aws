üìã Overview

Purpose: Consistent layout components and UI elements across the application

Business Context: This module provides reusable layout and UI components including system header, error boundaries, loading indicators, and theme controls to ensure consistent user experience throughout the application.

üéØ User Story Development Guide

üìù Specific US Templates

  • Consistent Header: As a user, I want to see a consistent header on all pages to maintain context
  • User Info Display: As a user, I want to see who I'm logged in as to confirm my identity
  • Logout Access: As a user, I want easy access to logout from any page for security
  • Error Handling: As a user, I want errors to be handled gracefully without crashing the app
  • Loading Feedback: As a user, I want to see loading indicators during data fetches
  • Theme Toggle: As a user, I want to switch between light/dark mode for comfort

‚ö° Development Acceleration Factors

  • SystemHeader: Pre-built header with logo, user info, and logout
  • ErrorBoundary: React error boundary for graceful error handling
  • LoadingSpinner: Reusable loading indicator component
  • ThemeToggle: Theme switcher with persistence
  • Layout Wrapper: Standard page layout structure

üìä Complexity Guidelines

  • Simple (1-2 pts): Use existing components, Display header, Show loading state
  • Medium (3-5 pts): Customize layout, Add error boundary, Implement theme toggle
  • Complex (5-8 pts): Create new layout patterns, Advanced animations, Accessibility enhancements

üí° Implementation Examples

Using SystemHeader

import { SystemHeader } from '~/components/layout/SystemHeader';

function MyPage() {
  return (
    <div>
      <SystemHeader 
        title="SAI Application"
        showUserInfo={true}
        showLogout={true}
      />
      <main>
        {/* Page content */}
      </main>
    </div>
  );
}

Error Boundary Usage

import { ErrorBoundary } from '~/components/layout/ErrorBoundary';

function App() {
  return (
    <ErrorBoundary
      fallback={<div>Something went wrong. Please refresh.</div>}
      onError={(error) => console.error('App error:', error)}
    >
      <Routes>
        {/* App routes */}
      </Routes>
    </ErrorBoundary>
  );
}

Loading Spinner

import { LoadingSpinner } from '~/components/layout/LoadingSpinner';

function DataPage() {
  const { data, loading } = useData();
  
  if (loading) {
    return <LoadingSpinner size="large" message="Loading data..." />;
  }
  
  return <div>{data}</div>;
}

Theme Toggle Integration

import { ThemeToggle } from '~/components/layout/ThemeToggle';

function HeaderControls() {
  return (
    <div className="controls">
      <ThemeToggle defaultTheme="light" />
      <LogoutButton />
    </div>
  );
}
